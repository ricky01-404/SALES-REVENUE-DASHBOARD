# Sales & Revenue Analytics Dashboard

An interactive business analytics dashboard built using Python, Streamlit, Pandas, and Plotly for analyzing sales and revenue data through dynamic KPIs, filters, and visualizations.

---

## Features

- CSV and Excel file upload support
- Multiple sample datasets
- Interactive KPI cards
- Revenue trend analysis
- Dynamic filters and slicers
- Automated business insights
- Interactive charts using Plotly
- Download filtered datasets
- Search functionality for raw data
- Modern dark-themed dashboard UI

---

## Technologies Used

- Python
- Streamlit
- Pandas
- Plotly Express
- OpenPyXL

---

## Dashboard Functionalities

### KPI Tracking

- Total Revenue
- Average Revenue
- Highest Revenue
- Growth Rate
- Total Records

### Data Visualization

- Revenue Trend Line Chart
- Revenue Distribution Pie Chart
- Category-wise Revenue Analysis
- Monthly Breakdown Charts

### Interactive Features

- File Upload System
- Dataset Switching
- Dynamic Sidebar Filters
- Date Range Filtering
- Download Filtered Data
